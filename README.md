# Proyecto React Principiante

Este repositorio contiene una app de práctica creada con React + Vite para aprender fundamentos paso a paso.

## Contenido

- app-react-principiante: aplicación principal con ejercicios resueltos.

## Ejecutar el proyecto

1. Entrar a la carpeta de la app:

```bash
cd app-react-principiante
```

2. Instalar dependencias:

```bash
npm install
```

3. Levantar entorno de desarrollo:

```bash
npm run dev
```

4. Build de producción:

```bash
npm run build
```

## Qué se practicó

- Estado con `useState`
- Eventos de botón e input controlado
- Renderizado condicional
- Lógica básica de UI (contador con límite, reset)
- Cambio dinámico de estilo (color del título)
