# Tienda de ropa en React

Aplicación sencilla hecha con **React + Vite** que muestra productos de ropa por categorías.

## Cómo iniciar

1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev
```

## Funcionalidades

- Catálogo de productos de ropa.
- Filtro por categoría.
- Vista simple y clara para practicar React.
